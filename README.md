QR Code Generator & Scanner 🔳


📖 Project Overview

This is a web-based QR Code Generator and Scanner application developed using HTML, CSS, and JavaScript.

The application allows users to generate QR codes from text, URLs, or images and also decode QR codes from uploaded images. It is designed with a modern UI and provides a smooth and responsive user experience.

✨ Features

🔳 QR Code Generation
->Generate QR codes from text and URLs
->Encode image data into QR format
->Instant QR preview
->Download QR code as PNG

📷 QR Code Scanning
->Upload QR code images for decoding
->Detect and display encoded content
->Supports:
  Text
  URLs (clickable links)
  Image data (preview display)
  
🎯 User Experience
->Drag & drop image upload
->Responsive design (mobile-friendly)
->Toast notifications for actions
->Clipboard copy support
->Clean and modern interface

🛠️ Technologies Used
->HTML5
->CSS3
->JavaScript
->QRious (QR code generation)
->jsQR (QR code decoding)
